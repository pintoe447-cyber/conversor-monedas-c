# Conversor de Monedas en C

## Descripción
Este proyecto consiste en una aplicación desarrollada en lenguaje C que permite realizar conversiones entre quetzales, dólares estadounidenses y euros.

## Prerrequisitos
- Tener instalado un compilador de C, como GCC
- Tener acceso a una terminal o consola
- Un editor de código como Visual Studio Code

## Pasos para su ejecución
1. Descargar o abrir el archivo del proyecto.
2. Compilar el código en la terminal.
3. Ejecutar el archivo compilado.
4. Ingresar la opción de conversión deseada.
5. Escribir la cantidad a convertir y visualizar el resultado.

## Ejemplo de compilación y ejecución
gcc conversor.c -o conversor  
./conversor
